Credit Insights Dashboard

Context

As part of a case study on credit products in the fintech industry, I designed and implemented a dashboard to analyze user eligibility, credit behavior, and financing adoption. The dashboard simulates how a fintech like Ualá could track credit KPIs, identify trends, and make data-driven decisions.

Objective

  • Highlights key credit KPIs across the user lifecycle.
  • Provides clear, actionable views for stakeholders.
  • Demonstrates the integration of BigQuery and Looker Studio within GCP.

Process

1) Data preparation

  • Modeled and transformed datasets with BigQuery SQL.
  • Normalized transactions, eligibility, and financing data into unified tables.
  • Created reusable queries to feed consistent metrics across the dashboard.

2) Dashboard design

Organized into three key tabs:

  • Transactional Behavior → Analysis of user spending patterns and credit usage.
  • Eligible Users → Trends in user eligibility and approval rates.
  • Financing → Insights into credit card financing adoption and repayment.
