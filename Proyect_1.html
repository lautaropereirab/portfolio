Project #1: DIA Supermarket Web Scraping

Description

An automated web-scraping process that collects structured information directly from DIA Online. The focus is on navigation automation and data extraction (categories, products, and prices), ensuring the script behaves like a user and stores results ready for analysis.

Scraping Flow

  1. Access & initial navigation: Selenium opens the site, expands the category menu, and navigates internal pages.
  2. Category exploration: Detects and iterates categories/subcategories, interacting with dynamic menus and checkboxes.
  3. Dynamic product loading: Auto-scrolls and clicks “Mostrar más” to load all visible items.
  4. Extraction with BeautifulSoup: Parses HTML to capture per product: name/descriptor, prices, discounts, price per kilo/unit, product link, and image.
  5. Structuring & storage: Consolidates data into a pandas DataFrame, adds a fecha_scrapeo field, and exports to CSV.

Output Schema

Main dataset columns (preserving original names):

  • descuento_1, descuento_2
  • campo_vacio_1, campo_vacio_2, campo_vacio_3
  • producto
  • precio_sin_descuento, precio, precio_por_k
  • link, imagen
  • fecha_scrapeo (añadida en cada ejecución)

Default output file: Productos Obtenidos/Almacen.csv

Libraries Used

  • Selenium – navigation & interaction automation
  • BeautifulSoup (bs4) – HTML parsing & extraction
  • Pandas – DataFrame handling & CSV export
  • lxml – fast, efficient parser
  • time / datetime / re – waits, timestamps, and regex
